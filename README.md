# Space Patches 🚀

A modern, space-themed web application showcasing European human spaceflight mission patches from 1978-2026.

## Features

- 🎯 Interactive grid of 142+ mission patches
- 🔍 Search functionality
- 🏢 Filter by agency/country
- 📅 Filter by year
- 🖼️ Beautiful modal view for patch details
- ⭐ Starfield background animation
- 📱 Fully responsive design
- ♿ Accessible interface

## About the Patches

This collection displays official mission patches worn by European astronauts during their spaceflight missions. The patches represent:

- **European Space Agency (ESA)**
- **Russian Agencies** (including Soviet Union missions with European astronauts)
- **International Partnerships** (ISS Expeditions)
- **NASA/ESA Collaborations** (Space Shuttle missions)
- **Commercial Partners** (SpaceX, Axiom Space)
- **CNES/ESA Missions** (French-European collaborations)

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with gradients and animations
- **JavaScript** - Dynamic filtering and interactivity
- **JSON** - Metadata management

## Data Source

Images are sourced from the official [ESA Spaceflight Patches Collection](https://www.esa.int/About_Us/50_years_of_ESA/European_human_spaceflight_patches).

## Usage

1. Browse all patches in the grid
2. Use the search bar to find specific patches
3. Filter by agency/country and year
4. Click any patch to view it in detail
5. Press ESC or click outside to close the modal
